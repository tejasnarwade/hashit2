## Animation Examples

### Insurance Checkbox Change
```
User clicks checkbox
        ↓
Checkbox state: ✗ → ✓
        ↓
Insurance info text fades in
        ↓
"Insurance active - Covers 70%" appears in green
```

### Slider Change
```
User drags slider 50% → 75%
        ↓
Investment Percentage updates: 50% → 75%
        ↓
Investment amount updates: ₹90,000 → ₹1,35,000
        ↓
Preview text updates instantly
```

### Year Submission
```
User clicks "END YEAR"
        ↓
Button shows "Processing..."
        ↓
Calculate all values (2-3 seconds)
        ↓
Show year result summary
        ↓
Leaderboard updates
        ↓
Button returns to "END YEAR"
```

## Interactive Elements

### Slider (0-100%)
- **Appearance:** Standard HTML range input
- **Color:** Blue (#3B82F6) when active
- **Width:** Full width of container
- **Min/Max Labels:** "Conservative" and "Aggressive"
- **Display Value:** Shows percentage above

### Checkbox (Insurance)
- **Size:** 1.25rem × 1.25rem
- **Color:** Green (#22C55E) when checked
- **Label:** Right-aligned text "Buy Insurance (5% premium = ₹X)"
- **Cursor:** Pointer on hover

### Button (End Year)
- **Text:** "END YEAR" or "Processing..."
- **Background:** Blue (#3B82F6) gradient
- **Text Color:** White
- **Hover:** Slightly darker blue
- **Disabled:** Grayed out when game is over
- **Width:** Full container width

  Font-weight: 600

  Font-weight: 400
