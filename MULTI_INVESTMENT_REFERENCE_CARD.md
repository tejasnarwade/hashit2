## Calculation Formula (Simple Version)

```
For Each Investment Option:

Amount = Capital × (Allocation % ÷ 100)
Return = Random % + Market Modifier
Gain/Loss = Amount × (Return % ÷ 100)

Example:
Capital: ₹180,000
Stocks: 25% allocation

Amount = 180,000 × (25 ÷ 100) = 45,000
Return = +8% (random) + 5% (modifier) = 13%
Gain = 45,000 × (13 ÷ 100) = 5,850

Result: Invested ₹45,000, gained ₹5,850
```

## Key Rules to Remember

```
✅ DO:
• Spread across options for diversification
• Review each option's risk level
• Adjust allocation based on your strategy
• Check results breakdown after each year
• Play multiple years to see patterns

❌ DON'T:
• Ignore returns (read them carefully)
• Forget total allocation must = 100%
• Expect same returns every year (random!)
• Miss the market modifier (affects all options)
• Neglect to check if you're still playing
```

After each year, you should see:
- ✓ 5 investment options with individual returns
- ✓ Each option shows gain or loss

- ✓ Total investment return calculated

## Quick Troubleshooting

| Issue | Solution |
|-------|----------|
| Cards not showing | Refresh page or restart server |
| Sliders not working | Clear browser cache, restart |
| Math looks wrong | Check event's market modifier |
| Game ending too soon | Make sure year allocation correct |
| Results missing | Wait for calculation to complete |
